Performance Achievement Evidence Management System
Business Process Automation using Google Apps Script
📌 Project Overview
พัฒนาระบบจัดการและจัดเก็บหลักฐานผลสัมฤทธิ์การปฏิบัติงานแบบดิจิทัล เพื่อเปลี่ยนกระบวนการทำงานจากแบบเดิม (Manual Process) ให้เป็นระบบอัตโนมัติ (Automation) ช่วยลดภาระงานเอกสารและสร้างระบบการจัดเก็บข้อมูลที่เป็นระเบียบ ตรวจสอบย้อนกลับได้แม่นยำ 100%

🚀 Key Features
Automated Workflow: วางลำดับขั้นตอนการส่งข้อมูลและจัดเก็บไฟล์โดยอัตโนมัติผ่าน Script

Centralized Data: รวมศูนย์การจัดเก็บหลักฐานไว้ในที่เดียว เพื่อความสะดวกในการเรียกดูและตรวจสอบ

Error Handling: มีระบบตรวจสอบความถูกต้องของข้อมูลก่อนการบันทึก เพื่อลดความผิดพลาดจากผู้ใช้งาน

Real-time Reporting: ดึงข้อมูลจากฐานข้อมูลมาสรุปผลได้ทันทีผ่าน Google Sheets

🛠️ Tech Stack
Platform: Google Apps Script (JavaScript-based)

Database: Google Sheets

Storage: Google Drive API Integration

🧠 Logical Thinking & Implementation
ในโปรเจกต์นี้ผมได้นำตรรกะทางคอมพิวเตอร์ (Logical Thinking) มาประยุกต์ใช้เพื่อแก้ปัญหาทางธุรกิจ ดังนี้:

System Workflow Design: ออกแบบ Flow การทำงานตั้งแต่ Input -> Processing -> Output ให้สอดคล้องกับความต้องการของผู้ใช้งาน

Database Management: ออกแบบโครงสร้างตารางข้อมูลให้สัมพันธ์กัน (Relational structure) เพื่อให้ระบบสามารถดึงข้อมูลรายงาน (Report) ออกมาได้อย่างรวดเร็วและถูกต้อง

Automation Logic: เขียน Script ควบคุมการสร้าง Folder และการตั้งชื่อไฟล์โดยอัตโนมัติเพื่อความเป็นสากลและหาข้อมูลได้ง่าย

📈 Results & Impact
ลดระยะเวลาในการจัดการและรวบรวมเอกสารลงอย่างมาก

เพิ่มประสิทธิภาพ (Efficiency) ในการติดตามผลงานขององค์กร

ข้อมูลมีความถูกต้องแม่นยำสูง และสามารถตรวจสอบย้อนกลับ (Traceability) ได้ทุกขั้นตอน
